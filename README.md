# Secure Chat App Simulation

## Overview

The Secure Chat App Simulation project aims to create a simple encrypted chat application that allows two users to communicate securely through a web browser or notebook environment.

---

## Team Roles

| Team Member | Role                                     | Focus
| ----------- | ----------------------------------       | ---------------------------------------------------
| Akshat      | Lead Security & Cryptography             | Implementing end-to-end encryption (E2EE) logic, key generation, and secure key exchange simulation.
| Patric      | Backend & Session Engineer               | Setting up the server/simulation environment, managing message routing between the two users, and handling session state.

## Project Goals

Create a secure chat application that:

* Allows two users to exchange messages
* Encrypts messages before sending
* Decrypts messages when received
* Demonstrates basic cybersecurity principles

---

## Development Steps

### Phase 1: User Interface

* Build message input field
* Build message output/display area
* Implement basic chat functionality

### Phase 2: Security Features

* Encrypt messages before sending
* Decrypt messages upon receipt
* Store message history

### Phase 3: Testing & Documentation

* Test application functionality
* Test encryption implementation
* Document findings and project outcomes
